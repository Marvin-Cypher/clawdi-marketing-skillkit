# Strict Cover System (Dev-Tool Editorial)

## Palette (only)
- `#0f0f0f` background
- `#1a1a1a` card
- `#333333` borders/lines
- `#ffffff` primary text
- `#999999` secondary text
- `#DE332C` accent (single focal concept only)

## Rules
- No gradients
- No glow
- No extra colors
- One red object max
- Clean left text / right diagram layout
- 4 major diagram elements max on cover
- Keep hierarchy:
  - title 64–72
  - subtitle ~24
  - diagram labels 14–18

## Prompt baseline
Use a strict prompt with explicit palette and “no gradient/no glow/no clutter” constraints.
